# User Funnel & Retention Analysis

## Project Overview

This project analyzes user behavior through a product funnel and retention metrics.

## Tools Used

- Python
- Pandas
- Matplotlib
- Jupyter Notebook

## Metrics Calculated

- Conversion Rate
- Drop-Off Rate
- Retention Rate
- Churn Rate

## Key Findings

- Conversion Rate: 5%
- Drop-Off Rate: 37.5%
- Retention Rate: 70%
- Churn Rate: 30%

## Business Recommendations

- Improve onboarding
- Reduce profile-completion friction
- Increase engagement
- Perform A/B testing
